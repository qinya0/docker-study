# docker-compose

```bash
# install docker-compose
1. open url: https://github.com/docker/compose/releases
2. find run.sh for one release, and download it
    wget https://github.com/docker/compose/releases/download/${release}/run.sh
3. mv run.sh /usr/local/bin/docker-compose
4. chmod 755  /usr/local/bin/docker-compose
5. you can use it
```
